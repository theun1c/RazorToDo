# RazorToDo
## Описание коммитов  
| Название | Описание                                                             |  
| -------- | -------------------------------------------------------------------- |  
| docs     | обновление документации (readme)                                     |  
| feat     | добавление нового функционала                                        |  
| fix      | исправление ошибок                                                   |  
| test     | добавление unit-тестов к проекту или тестовой документации           |
